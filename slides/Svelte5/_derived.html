Les variables dérivées

--- layout: default transition: slide-up ---

$derived

La rune $derived de Svelte 5 permet de déclarer une variabe dérivée. C'est à dire une variable dont la valaur dépent d'une autre variable.

Avec Svelte 4:

```svelte

double = {double}

```

Avec Svelte 5:

```svelte

double = {double}

```

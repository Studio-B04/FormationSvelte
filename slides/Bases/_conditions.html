7. Les conditions

--- layout: default transition: slide-up ---

Les conditions dans les templates Svelte

Si l'on souhaite afficher un élement en fonction d'une condition booléene, il faut utiliser la syntaxe :

```svelte {*}{lines:false} {#if expression} ... {/if} ```
```svelte {#if count > 10}

Vous avez cliqué plus de 10 fois

{/if} ```
--- layout: default transition: slide-up ---

Les conditions dans les templates Svelte

Si on souhaite gérer le cas contraire il faut utiliser la syntaxe :

```svelte {*}{lines:false} {#if expression} ... {:else} ... {/if} ```
```svelte {#if count >= 10}

Vous avez cliqué au moins 10 fois

{:else}

Vous avez cliqué moins de 10 fois

{/if} ```
--- layout: default transition: slide-left ---

Les conditions dans les templates Svelte

Comme en javascript, il est possible de gérer plusieurs cas avec la syntaxe :

```svelte {*}{lines:false} {#if expression} ... {:else if} ... {:else} ... {/if} ```
```svelte {#if count >= 10}

Vous avez cliqué au moins 10 fois

{:else if count >= 5}

Vous avez cliqué au moins 5 fois

{:else}

Vous avez cliqué moins de 5 fois

{/if} ```

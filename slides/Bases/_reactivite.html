6. La réactivité en Svelte

--- layout: default transition: slide-up ---

La réactivité en svelte.

Contrairement à d'autres frameworks comme React, les variables déclarées dans un composant Svelte sont par nature réactives. Démonstration :

```svelte

Vous avez cliqué {count} fois sur le bouton.

```

Essayons sur le Playground de Svelte !

--- layout: default transition: slide-up ---

La réactivité en svelte.

En revanche, si une variable est calculée par rapport à une autre variable, alors sa valeur ne sera pas mise à jour :

```svelte

Vous avez cliqué {count} fois sur le bouton.

Double = {double}.

```

Essayons sur le Playground de Svelte !

--- layout: default transition: slide-up ---

La réactivité en svelte.

Pour rendre la variable double réactive et donc changer sa valeur dès que count change, il faut utiliser le $:

```svelte

Vous avez cliqué {count} fois sur le bouton.

Double = {double}.

```
> Attention ! > Les variables déclarées avec $: demandent beaucoup de mémoire et peuvent altérer la performance globale de votre application si elle sont utilisées trop souvent.

Essayons sur le Playground de Svelte !

La réactivité en svelte.

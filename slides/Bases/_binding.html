9. Le binding

--- layout: default transition: slide-up ---

Le binding

Les données circulent généralement du parent à l'enfant. La directive bind: permet aux données de circuler dans l'autre sens, de l'enfant au parent.

Le mot clé bind permet de faire correspondre un attribut HTML à une variable, par exemple ce code peut facilement être simplifié :

```svelte

value = {value ?? 'inconnue'}

```
--- layout: default transition: slide-up ---

Le binding

Avec le mot clé bind: :

```svelte

value = {text ?? 'inconnue'}

```

Encore plus simple si le nom de la propriété et sa valeur sont identique :

```svelte

value = {value ?? 'inconnue'}

```

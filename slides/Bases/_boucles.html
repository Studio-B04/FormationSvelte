8. Les boucles

--- layout: default transition: slide-up ---

Les boucles dans les templates Svelte

Pour iterer sur des valeurs en Svelte, il faut utilser la syntaxe

```svelte {*}{lines:false} {#each items as item} ... {/each} ```

Prenons un exemple :

```svelte
    {#each users as user}
  1. {user}
    2. {/each}
```

Les boucles dans les templates Svelte

Si on souhaite récupérer l'index de la chaque élément de la boucle, il faut passer un deuximème paramètre :

```svelte {*}{lines:false} {#each items as item, index} ... {/each} ```

Dans notre exemple précédent :

```svelte
    {#each users as user, index}
  • {index + 1} - {user}
    • {/each}
```

Les boucles dans les templates Svelte

En Svelte il est possible très simplement de gérer le cas ou l'objet itérable est vide :

```svelte {*}{lines:false} {#each items as item, index} ... {:else} ... {/each} ```

Dans notre exemple précédent :

```svelte
    {#each users as user, index}
  • {index + 1} - {user}
    • {:else}
  • Il n'y a aucun utilisateurs
    • {/each}
```

Les boucles dans les templates Svelte

il peut-être parfois indispensable dans les boucle d'identifier chaque élément par une clée unique.

En React il existe l'attribut key à placer sur le premier élément de la boucle. En Svelte, c'est plus simple, on précise simplement la clé entre parenthèse dans la boucle :

```svelte {*}{lines:false} {#each items as item (key)} ... {/each} ```
```svelte
    {#each users as user, index, (user.firstname)}
  • {index + 1} - {user.firstname} {user.lastname}
    • {/each}
```

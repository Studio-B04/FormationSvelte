C'est quoi un adapter ?

Lorsqu'une application est développée avec SvelteKit, elle doit être déployée sur un environnement spécifique (Node.js, serveur statique, plateforme cloud, etc.) afin de la rendre publique.

Les adapters sont des modules qui transforment une application SvelteKit afin qu'elle fonctionne sur une plateforme cible.

Lorsqu'une application est buildée avec la commande npm run build, le code est compilé et organisé selon des patterns prédéfinis par la plateforme sur laquelle vous projetez de déployer votre projet.

L'adapter va alors créer un dossier de sortie (par défaut, ce sera le dossier /build) qui contiendra tous les fichiers nécéssaires pour permettre à votre application d'exécuté sur le serveur de production.

Fonctionnement dans SvelteKit

La configuration de l'adapter se fait dans le fichier svelte.config.js :

```js import adapter from '...'; export default { kit: { adapter: adapter() }, }; ```
